【 Facebook 为 Graph Search 增加新功能，可搜索消息帖、评论、签到等】Graph Search 是 Facebook 在今年一月份推出的社交图谱搜索工具，用户使用 Graph Search 时能以自然语言搜索出相关的人、地点和事物。Facebook 用户产生内容的信息流（News Feed）、社交行为产生的动态信息（Timeline），以及通过支持 Open Graph API 的第三方网站嵌入的 Like 按钮分享导入 Facebook 站内的各种信息构成了 Graph Search 搜索的主要信息来源。如今，Facebook 宣布为 Graph Search 增加新功能，用户可以通过 Graph Search 搜索到其他用户的消息帖、评论、签到等信息，用户过去和现在的信息都将“一览无余”。总而言之一句话，Graph Search 将会让你的隐私变得越来越透明。目前，这一功能首先向美国地区的部分英语用户开放。

【 Google 推出 Web Designer，为搭建 HTML5 互动网站和广告提供可视化工具】今年 7 月份的时候，Google 曾暗示将推出 Web Designer，从而为搭建 HTML5 互动网站和广告提供可视化工具，但之后却再无消息。如今，沉寂了几个月后， Web Designer 终于面世，推出了 beta 版，Mac 用户和 Windows 用户都可以下载。根据 Google 现在的描述，Web Designer 将帮助广告商轻松创建移动端和桌面端的 HTML5 广告。

【大数据安全公司 Dataguise 获 1300 万美元投资】DataGuise 是一家提供大数据安全解决方案的初创企业。其 DgSecure 数据保护产品系列可检测企业包括云部署在内的整个网络的数据安全。系统会定位和识别企业内部各个数据容器、微软 SharePoint 网站、NFS（网络文件服务器）及 Hadoop 集群中的敏感数据，并提供数据泄露的风险评估，帮助企业制定相应的补救措施（如数据屏蔽或数据隔离等），从而让企业将数据安全地共享给员工和外部第三方。这家公司刚刚完成 1300 万美元的 B 轮融资，由 Toba Capital 领投。迄今为止，其总融资额已达 2030 万美元。

【国家互联网信息办查处一批违规移动新闻客户端】国家互联网信息办公室昨天发出通报，对“抽屉新热榜”、“鲜果联播”、“ZAKER 新闻阅读”、“3G 门户新闻”、“一五一十部落”、“蜜蜂新闻”等一批未依法取得互联网新闻信息服务资质、违规从事互联网新闻信息服务的移动客户端，依法要求限期整改。《通报》指出，移动新闻客户端运营单位从事互联网新闻信息服务必须经互联网信息内容主管部门审批备案，并依法取得相应资质，未取得资质不得从事互联网新闻信息服务，各移动应用商店也不得为违法违规移动客户端提供上架发布、推荐下载等服务。

【去哪儿提交 IPO 招股书，拟筹资 1.25 亿美元】去哪儿网周一向美国证券交易委员会（SEC）提交了 IPO 招股书。招股书显示，去哪儿计划通过 IPO 筹集最多 1.25 亿美元资金，并计划在纽约证券交易所上市，股票交易代码为“QUNR”。高盛集团、德银证券、Stifel、Nicolaus & Company、太平洋寰冠证券和香港华兴资本证券将担任去哪儿 IPO 交易的承销商。
招股书披露上市之前，去哪儿最大机构股东百度持股 61.05%, 创始人庄辰超持股 7.23%。2011 年 6 月，百度与去哪儿达成深度战略合作协议，百度向去哪儿注入 3.06 亿美元战略投资，成为去哪儿第一大机构股东。
