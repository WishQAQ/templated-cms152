微软发布 Windows 8.1 广告，突出 Start 按钮

【微软发布 Windows 8.1 广告，突出 Start 按钮】微软已经通过官方 YouTube 频道发布了 Windows 8.1 的首个广告。在广告中，微软突出了 Windows 8.1 的“Start”按钮、进一步订制的“Start”屏幕，以及在同一屏幕上运行两个应用的 Snap View 模式。广告时长 30 秒，很可能会投放至电视。此前，微软在 Windows 8 中取消了 Start 按钮，引发用户不满。不过目前仍有很多 Windows 用户尚未升级至 Windows 8，因此他们可能会直接接触 Windows 8.1。

【 Facebook 调整“News Feed”广告匹配算法】Facebook 宣布已经改进“News Feed”广告投放算法，为用户更精准地过滤不相关广告，还特别强调了新的算法将更注重于用户对于 News Feed 广告的反馈，比如用户的举报和隐藏相关广告的频率。
“News Feed”是 Facebook 今年三月更新的一种全新的内容筛选方式，会针对不同的内容源为用户匹配不同的呈现方式，试图为用户带来“个性化报纸”的体验。这次算法的改进将改善为用户匹配广告的精准性以及广告质量。Facebook 表示，算法的改进对广告商和普通用户都有好处。对广告商来说，这意味着更加精准的广告投放；而对于普通用户而言，广告将变得更具有相关性并且那些没兴趣的广告将减少，一句话总结就是“在对的时间投放出对的广告”。

【 Twitter 推出“Websites”流量分析面板，为网站提供数据分析】Twitter 在今年六月推出了“Analytics”功能板块，包括“Timeline activity”和“Followers”两个下拉功能，为 Twitter 用户的帐号提供数据分析服务，比如用户的 tweet 有多少人看过或者粉丝的增长趋势等。而这次的“Websites”也属于“Analytics”中的下拉功能之一，不同的是“Websites”并非为 Twitter 用户的帐号服务，而是为 Twitter 导向的网站提供流量分析功能，这些网站导向是通过用户在自己主页植入相关代码实现的，这和 Google Analytics 追踪功能类似。可以说，这一功能几乎就是为广告商定制的。

【无线充电企业 PowerbyProxi 获三星 400 万美元投资】根据美国科技博客 AllThingsD 的报道，无线充电初创企业 PowerbyProxi 在第三轮融资中，获得了三星旗下风险投资公司三星资本（Samsung Ventures）的 400 万美元投资。PowerbyProxi 的无线充电系统允许多台设备同时在一个充电器上进行充电，且在设备的摆放位置方面更具灵活性。根据投资协议，三星资本（Samsung Ventures）美洲地区高级投资经理 Michael Pachos 将加入 PowerbyProxi 公司董事会，同时两家公司将达成战略合作伙伴关系。

【百度积极筹备互联网金融，意欲效仿阿里推出基金产品】这是阿里旗下余额宝的成功所引发的一场蝴蝶效应。据悉，百度近期正密切接触包括华夏基金在内的数家公司，商谈定制基金产品，且百度此次定位清晰，要推出一款比余额宝更棒的产品，并且只能在百度平台上销售，与余额宝要有一定差异化，年回报要达到 8%。就在几天前，百度旗下的“百度金融”、“百度理财”、“百度证券”三款金融商品官方微博已悄然认证上线，外界推测百度金融将于 10 月正式上线。
事实上，百度今年以来一直在积极布局互联网金融业务。早在今年 7 月，百度旗下的支付工具“百付宝”获得了第三方支付牌照，8 月底，百度金融测试版也悄然上线。进入 9 月，百度方面又证实，百度在上海设立实体小额贷款公司申请已获批准，将落户嘉定。
